# Crownless-A-Text-Based-RPG-
A beginner-friendly text-based RPG written in Java as a learning project. Players can choose from 3 classes, each with unique traits. The story follows a small adventure where the player must save a village from internal threats while the crown is distracted by an ongoing war. Focused on OOP and game design basics.
